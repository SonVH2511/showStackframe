### README

An IDA plugin that helps show the stack frame base relative to the current RSP. 


<img width="2061" height="1557" alt="image" src="https://github.com/user-attachments/assets/94d77fd1-21c4-45e4-8935-59d6592eccb9" />

### Requirement

Supports for 9.0+ version.


### Usage

Just throw file stack_viewer.py to plugins folder. Shortcut to open:`ctrl + shift + K`.

Plugins path: `%AppData%\Hex-Rays\IDA Pro\plugins`.
